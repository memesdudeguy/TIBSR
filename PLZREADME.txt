TIBSR, a web browser interface that has a nice user-friendly gui, cursor, and works with DoorsCS7 shell
to run first send the .8xp Binary(this will be your Browser) to your calc with Tilp, TI connect, or TI connect CE.
then install Python3 or later.
once Python is installed run the batch file, then run the py script to make sure it's working.
now install usb seral driver using zadig should be in Install driver folder.
once you get the driver successfully installed open, and connect your calc to the pc.
now open py script first, and run your browser.
finally you should get your calc on the internet(py terminal will get "calc connected")
